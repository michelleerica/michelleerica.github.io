The brief

In a group, build a web application in ruby on rails using at least 3 models and RESTful routes, utilising APIs, ruby gems and ajax

The idea

Think StackExchange, Quora or Yahoo Answers but with a location-specific spin to it.
Location-based Q&A web application whereby users can ask other users location specific questions. The app is aimed at connecting the curious with the knowledgeable within a particular geographical area, with language translating available for non-english speakers.

The stack

HTML / CSS / Javascript / jQuery / Ruby on Rails / semantic ui / Google Maps api / yandex translate api

Debrief

Fraga was a great opportunity to work as part of a team with developers with different strengths and skills to my own. As a result, I gained a lot of insight into how other people work and think through problems, and how to deal with merge conflicts in git! Throughout the project, the team kept track of tasks and progress using Trello and daily stand ups to make sure we were across what each other was doing, and were all working towards a common goal. I’m really proud of what we accomplished in a week.
